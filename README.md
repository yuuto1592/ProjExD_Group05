# ごちゃまぜRPG

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
マップを探索し、さまざまな試練を乗り越えて最奥に潜むボスを撃破するギミック「ごちゃまぜ」のRPG！
* 2つの戦闘パターン：敵とのバトルは、コマンド選択式の「RPGバトル」と、足場が崩れていく「タイルゲーム」の2種類がランダムで発生！
* マップ探索とイベント：踏むと大ダメージを受ける危険な罠を避けつつ、ミニゲームイベントをクリアしてHPを回復しよう！
* ラストバトル（ボス戦）：最後に待ち受けるボスの凶悪で激しい弾幕をよけきり、見事撃破すればゲームクリア！「ノーダメ」でボスを撃破できたら伝説だ！！！

## ゲームの遊び方
### 1.マップ移動
* 移動：矢印キー（↑ ↓ ← →）
* マップを探索して敵やボスを探す。紫色の「罠」を踏むとダメージを受けるので注意。

### 2.通常戦闘
#### RPGバトル
* 選択：矢印キー（↑ ↓）
* 決定：Enterキー

#### タイルゲーム
* 移動：矢印キー（↑ ↓ ← →）
* 崩れ落ちる床から顔を出す敵を避けながら、アイテムを集めてクリアを目指す。

### 3.ミニゲーム
* 接触するとランダムなミニゲームが始まり、クリアするとHPが回復する。
#### 連打ゲーム
* スペースキーを制限時間以内に指定回数連打

#### 回避ゲーム
* 矢印キー（← →）で、上から落ちてくる障害物を回避

### 4.ラスボス戦（弾幕ゲーム）
* 移動：矢印キー（↑ ↓ ← →）またはWASDキー　※同時押しで移動速度をアップできる
* 攻撃：スペースキー
* ボスの凶悪で激しい弾幕を避けながら、攻撃を当ててボスのライフを削り切れば勝利<br>
※ボス戦開始時のHP10につき、ハート1個に換算される（どれだけHPが低くても、最低1個は保証される）。


## ゲームの実装
### 共通基本機能
* マップを搭載したRPG風のもの

### 分担追加機能
* 戦闘イベント2パターン（担当：ハコザキ、カゲヤマ）<br>
　カゲヤマ：足場がランダムに崩れていくミニゲームを作成。アイテムを最初に出現した数の60％分獲得できればクリア。足場の下にいる敵に落ちたら一発アウト<br>
　ハコザキ：敵をランダムな種類にする。
* ボス戦（担当：タドコロ）:ボス戦の弾幕ゲームを作成。ボスマスを踏むと弾幕ゲームが即座に始まる。WASD, 矢印キーで移動（組み合わせで加速）スペースキーで攻撃が可能。
* ミニゲーム（担当：ハヤシ）:該当のマスを踏むとランダムでスペースキー連打か弾幕避け(簡単)のミニゲームに移行する機能を実装+クリアしたらplayerのHPを全回復
* 罠（担当：タカハシ）:紫色のマスを踏むと罠が発生し1回踏むごとに30ダメージ食らう
* テレポート(担当:ハヤシ) :該当のマスを踏むとマップが切り替わる
