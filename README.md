# ごちゃまぜRPG

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* マップを探索し敵、ボスを倒すゲーム
* ボスの弾幕をよけて撃破すればゲームクリア

## ゲームの遊び方
* マップを移動（矢印キー）して敵と戦う
* ラスボスは弾幕ゲーム（矢印キー、WASDで移動、スペースで攻撃）

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画

### 分担追加機能
* 戦闘イベント2パターン（担当：ハコザキ、カゲヤマ）
* ボス戦（担当：タドコロ）
* ミニゲーム（担当：ハヤシ）
* 罠（担当：タカハシ）

### メモ
* 色は定数として定義してから使う
* 定数、変数を定義するときは何に使用するか明確に示す
* classを定義するときは{イベント名}{class名}で定義する
