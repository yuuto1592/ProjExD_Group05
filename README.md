# ごちゃまぜRPG

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* マップを探索し敵、ボスを倒すゲーム
* 敵ごとにギミックが違う
* イベントを駆使してボス戦を楽に戦う
* 罠を上手くよけながらボスを探す
* ボスの弾幕をよけて撃破すればゲームクリア

## ボス戦
![titile](readme_img/boss_battle.png)

## ゲームの遊び方
* マップを移動（矢印キー）して敵と戦う
* ラスボスは弾幕ゲーム（矢印キー、WASDで移動※同時押しで速度調整可能、スペースで攻撃）

## ゲームの実装
### 共通基本機能
* マップを搭載したRPG風のもの

### 分担追加機能
* 戦闘イベント2パターン（担当：ハコザキ、カゲヤマ）
　カゲヤマ：足場がランダムに崩れていくミニゲームを作成。アイテムを最初に出現した数の60％分獲得できればクリア。足場の下にいる敵に落ちたら一発アウト
　ハコザキ：敵をランダムな種類にする。
* ボス戦（担当：タドコロ）:ボス戦の弾幕ゲームを作成。ボスマスを踏むと弾幕ゲームが即座に始まる。WASD, 矢印キーで移動（組み合わせで加速）スペースキーで攻撃が可能。
* ミニゲーム（担当：ハヤシ）:該当のマスを踏むとランダムでスペースキー連打か弾幕避け(簡単)のミニゲームに移行する機能を実装
* 罠（担当：タカハシ）:紫色のマスを踏むと罠が発生し1回踏むごとに30ダメージ食らう
